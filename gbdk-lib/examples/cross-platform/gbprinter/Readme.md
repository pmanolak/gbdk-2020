
GBPrinter
=========

Prints the image onto the game boy printer. To connect the game boy printer to the 
Mega Duck or the Sega Game Gear you need to solder the specific link cable adapter.

Sega Game Gear cable pinout:

DPC0 <--> SOUT
DPC1 <--> SIN
DPC6 <--> CLK
GND  <--> GND
